AI TRPG用説明テキスト

・使い方
このテンプレートは、AIにKPとしての基本方針を守らせるための共通補助文です。
template.txt をテキストファイルとして、シナリオと同時にAIへ読み込ませてください。
readme.txtをAIに読み込ませる必要はございません。（これは皆様に向けた注釈です。）
AIの「親切行動」や、過剰な補完を制御する目的で作成しています。

ただし、template.txt はシステムごとの判定方式そのものを決めるものではありません。
使用するTRPGシステムやシナリオ本文に応じて、判定方式や成功条件は別途追加で指定してください。

・AIに追加で伝えてください
システムによって、ロールオーバー／ロールアンダーなど判定方式が異なります。
また、未公開の独自システムである可能性もあるため、template.txtと同時にAIに判定方式を明示してください。

例：
このシステムは目標値以下で成功です
このシステムは目標値以上で成功です
対抗判定があります
クリティカル／ファンブルがあります
判定結果の段階成功があります

・なぜ template.txt が必要か
AIは何も宣言しない場合、こちらに対して過剰な配慮を行うことがあります。
例えば、「1d100を振ってください」と命令した場合でも、完全なランダムではなく、成功しやすい数値を出そうとすることがあります。

たとえば CoC で、キャラクターが目星70を持っている場合、1～70の範囲だけを出してしまいます。
ですが、失敗して言い訳を考えたり、そこから展開が変わるのもTRPGの楽しみです。
そのため template.txt では、「失敗しても進行を止めない」「判定はランダムに行う」といった方針を明記しています。

・なぜこのtxtをGitHubに？
更新頻度が高くなることが予想されるため、template.txt は更新しやすい GitHub で公開することにしました。
また、私の知らないマイナーシステムや独自TRPGへの流用も想定しており、これらすべてのTRPGへ一人で対応するのは難しいと考えたためです。
そのため、このテンプレートは複数人で改善していける形を目指しています。
動作上の問題や改善案、独自システムでの補足提案などがありましたら、Issue や Pull Request でお知らせください。

・特筆事項
ロールオーバー／ロールアンダーはシステムごとに異なるため、template.txt を貼る際に必ず宣言してください。
